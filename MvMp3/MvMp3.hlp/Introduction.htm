Move Mp3 files Into One Directory

Introduction

The purpose of the application is consolidate the mp3 files for several disks into a single directory with names that sort in the order of the story. When the files are saved they are saved into separate directories, one for each disk in the set. This application moves the files into the target directory, which is the directory that holds the directories for the disks.

There are only two relevant commands:

  • Specify Target or Parent Directory -- The directory that holds the disk directories
  • Specify a Prefix (i.e. Alphanumeric) and move all the files in directories below the target directory.
